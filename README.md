# Project Description

The **SaaS in Business Management System** is an AI-powered WordPress plugin designed to help businesses manage operations more efficiently through intelligent automation and data analysis.

This system provides business owners with tools for inventory management, expense tracking, sales monitoring, and AI-generated business insights. The plugin aims to simplify business operations, reduce manual work, and improve decision-making using artificial intelligence.

The project is being developed as a semester project for the CSE4104 course.

# System Architecture

The system follows a layered architecture consisting of:

* Frontend Layer (WordPress Dashboard)
* Backend Layer (PHP Plugin Logic)
* Database Layer (MySQL)
* AI Integration Layer
* External AI Service (OpenAI/Gemini API)

Architecture diagrams are available in the `diagrams/` directory.
